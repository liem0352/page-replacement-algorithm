# 页面置换算法实验

## 项目简介
本项目实现了操作系统中两种经典的页面置换算法：FIFO（先进先出）和 LRU（最近最久未使用），使用C语言实现，包含完整的测试脚本和实验文档。


## 算法说明

### FIFO（先进先出）算法
FIFO算法是一种简单的页面置换算法，总是淘汰最先进入内存的页面，即选择在内存中驻留时间最久的页面予以淘汰。

### LRU（最近最久未使用）算法
LRU算法基于局部性原理，淘汰最近最久未被访问的页面。为每个页面设置访问时间戳，每次访问时更新，置换时选择时间戳最小的页面。

## 使用说明
1. 输入页面引用串的长度
2. 输入页面引用串（用空格分隔）
3. 输入内存帧数
4. 选择算法：1-FIFO，2-LRU，3-比较两种算法，0-退出

## 示例
```
页面引用串：1 2 3 4 2 1 5
内存帧数：3
```
程序会显示每一步的页面访问过程，并统计缺页次数和缺页率。

## 性能分析
通过对比两种算法的缺页次数和缺页率，可以分析不同页面置换算法的性能差异。一般来说，LRU算法的性能优于FIFO算法，因为LRU考虑了页面的访问历史。
